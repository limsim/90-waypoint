# 90 Waypoint Walk

A browser-based visualiser for the [90 Waypoint Walk](https://www.marcusjohnhenrybrown.com/the-90-waypoint-walk/) — a walking experiment created by Marcus John Henry Brown. The original walk was a 19 km stroll through Munich in August 2016, navigated by following a notebook of 90 left/right turns.

This app draws the walk as an orthogonal grid map. The turn sequence is fixed (matching the original Munich walk), but segment lengths are randomised on each generation, so every map looks different while following the same underlying route.

## How it works

The walker starts at the centre of the canvas facing North. At each of the 90 waypoints, a turn from the fixed sequence is applied (Left = 90° counter-clockwise, Right = 90° clockwise), then the walker moves a random distance in the new heading. Ten **wildcards** distributed through the sequence skip the turn — the walker continues straight ahead instead.

After all 90 waypoints are placed, the path is centred on the canvas. If a segment would go out of bounds, the app tries alternative directions before shortening the segment.

## Getting started

**Prerequisites:** Node.js and Python 3.

```bash
npm install
npm run build
npm run serve
```

Open http://localhost:8000 in your browser.

During development, run the TypeScript compiler in watch mode in a separate terminal:

```bash
npm run dev
```

Changes to `src/index.ts` are compiled automatically; refresh the browser to see them.

## Controls

| Control | Description |
|---|---|
| **Generate Walk** | Draw a new map using the fixed turn sequence with fresh random segment lengths. |
| **Clear** | Remove all waypoints and lines. |
| **Min Distance** | Slider (20–100 px, default 60 px). Sets the minimum segment length for the next generation. |
| **Show Wildcards** | Toggle the gold rings that mark the 10 wildcard waypoints. |

Clicking a waypoint shows a tooltip with its number, turn direction, and cumulative distance from the start. Hovering highlights the waypoint and its adjacent path segments.

## Visual encoding

| Visual | Meaning |
|---|---|
| Black circle | Waypoint 1 (start) or waypoint 90 (end) |
| White circle | All other waypoints |
| Gold ring | Wildcard — walker went straight instead of turning |
| Number | Waypoint sequence (1–90) |
